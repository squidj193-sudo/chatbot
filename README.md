## 環境變數說明

請自行建立 `.env` 檔案，並填入自己的 API key。

範例：

```env
GEMINI_API_KEY=your_api_key_here
```

## 遇到的問題與解法

### 問題 1

問題：git指令無法使用
解法：
詢問AI，告知為git之path未加入環境參數，加入後即可正常使用。

### 問題 2

問題：API key使用次數告罄
解法：
更換API key。

### 問題 3

問題：AI詢問次數用完
解法：
更換AI模型，由Opus4.6 改為Gemini 3.1。
